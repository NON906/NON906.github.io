遊び方

開始までの準備

  1. PCなどで以下のURLから、マーカーボックスをダウンロードして印刷をし、組み立ててください。

    (URL: http://mumei.rosx.net/sp/arpong/ArBox.pdf)

    ダウンロード(pdf)

  2. タイトル画面からStartを押し、Player1かPlayer2のどちらかを選択してください。
  3. （Player2のみ）Player1が使っているスマートフォンかタブレットを選択してください。
  4. 「Please project this marker.」と表示されたら、マーカーボックスをスマートフォンかタブレットのカメラで写してください。
    （もし、画像がぼやけてしまって、うまく反応しない場合は、Focusボタンでフォーカスを合わせてみてください）
    （再度「Please project this marker.」が表示された場合も再度カメラで写してください）
  5. もしラケットが見えない場合は、カメラをマーカーボックスから少しだけ離してください。

遊び方

スマートフォンかタブレットを動かすと、それに合わせてラケットが動きます。
ラケットを動かし、飛んでくるボールを打ち返してください。
終了したい場合はスマートフォンかタブレットの戻るキーを押してください。
