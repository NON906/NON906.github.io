# XRViewer

VR配信や動画配信が出来るアバター配信アプリケーション、およびそのビュワーアプリケーションです。  
以下の機能が存在します。
- VR空間で撮影し、それをRTMPでライブ配信（Youtube Liveなど）
- スマートフォンVR（Cardboardなど）やMeta Quest2で配信者のいるVR空間に入ることができる
- PCゲーム画面のウィンドウなど、配信者のデスクトップ画面をVR空間や動画で共有

※現在テスト版です
